# Ejemplos React  

Este es un proyecto de frontend desarrollado con **React** y **JavaScript**, utilizando **Vite** como herramienta de desarrollo.  

---

## Descripción  

Este repositorio contiene ejemplos básicos para aprender y practicar el uso de React, incluyendo la configuración inicial del entorno y la ejecución del proyecto.  

---

## Instalación  

### 1. Clonar el repositorio  
```bash
git clone "url"

## Revisar si tenemos instalado o la version de nuestro node y el npm  

2.Revisar si tienes instalado node con el siguiente comando **node -v**  

3.Revisar si tienes instalado npm con el siquiente comando  **npm -v**  

## Crear un proyecto  

4.Ingresar el siguiente comando **npm create vite@5.1.0 nombre-app -- template react  

## Entrar a la carpeta del proyecto  

5.Ingresar el siguiente codigo **cd nombre-app**  

## Instalar npm  

6.Ingresar el siguiente comando **npm install**  

## Correr el codigo  

7.Ingresar el siguiente codigo **npm run dev**  

### Instalar una api de mapas de google

8.Ingresar el siguiente codigo **npm i @react--google-maps/api**
